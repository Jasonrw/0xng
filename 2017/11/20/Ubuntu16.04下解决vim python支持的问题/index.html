vim

问题

准备配置vim成python开发环境，参考文章：VIM and Python - a Match Made in Heaven,但是配置vim成python support时发现无论如何重装，还是自己去github上下载源码自己编译都不行，最后的结果都是
+python3 -python

探索

在google之后发现在Ubuntu16.04下面并不能同时python support和Python3 support，（参考：StackOverflow）,这才发现原因。

解决

直接show you the code:

只安装python的支持，当然你也可以只安装python3的支持，不过那样是没有意义的，因为直接
sudo apt install vim
