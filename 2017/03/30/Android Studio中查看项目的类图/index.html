Android Studio中查看项目的类图

android开发
  • android Studio
  • android开发
  • 类图
  • 阅读代码

类图作为详细设计中非常重要的一块，对于理解整个项目的结构异常重要，在阅读开源程序时，如果有类图的辅助，也可以事半功倍。
Visual Studio中就自带了根据项目生成类图的功能，最近在做安卓App开发的时候，从GitHub上Clone了一个开源的项目，奈何看起来异常艰难（面向对象层层嵌套有木有啊，连抽象都不知道抽到哪里去了…），于是Google了一下AS上关于生成类图的解决方案。

探索

Google上成片都是采用在AS中安装SimpleUMLCE（CE代表的是社区版）插件的方法来生成类图，但是按照里面所描述的，在File–>setting–>plugin–>browsRepositories里并搜索不到该插件。发现下面还有一个install plugin from disk，抱着试一试的态度上intelliJ IDEA Plugins Repository，搜了simpleUML，发现果然可以下载，点击查看详细页面，也可以直接下载

解决方案

1. 下载插件

【插件详情页】【直接下载】

2. 安装插件

2.1 打开Android Studio，选择File–>setting–>plugin, 找到install plugin from disk按钮

2.2 选择simpleUMLCE_8205.jar,安装完毕就OK了。

3. 使用方法

3.1 找到你需要生成类图的包或者项目（比如app）文件夹，右键单击，选择Add to simpleUML diagram, 选择New Diagram...,然后选择保存的位置（可以随便选，我选的项目的根目录），保存好后选择AS左边的SimpleUML选项卡

就可以看到生成好的类图了，(但所有的类都是折叠的，不知道有木有办法能够一键全部展开，知道的小伙伴麻烦发邮件给我…感激不尽！)另外右侧也有一个选项卡，可以打开动态的类图，信息比较少，但是看起来更加清爽。

注意事项：该插件运行基于AS自身的编译器，故必须在项目能通过编译的情况下才能生成正常的类图。
