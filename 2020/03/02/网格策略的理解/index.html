网格策略的理解

前几年做过量化择时，主要帮人实现策略，从来没有想着有策略我自己能够理解，第一次看到网格策略的时候，可能是两年以前的事了，才发现，原来有点东西只要写的清楚，我还是能够理解的

什么叫交易策略

我对证券类（股票、基金、期货、外汇）的投资的理解其实有经历过几个阶段

第一个阶段 选股

在第一次接触到证券类的产品后，第一个想法一定是股神现在都买的什么样的股票，我也要跟他买一样的，这个阶段还没有真正的参与到证券交易这个过程中去，以为选好了一支（或几支）股票以后就坐等收益就行了

第二个阶段 追涨

在开始正式的股票交易以后，因为第一个阶段没有完全弄清楚怎么选股（事实上根本不可能弄清楚），故而在选股的时候只能借助于自己对于股票市场的理解，看到有涨的势头就赶紧跟，结果可想而知。

第三个阶段 追跌

经过第二个阶段，逐渐明白了应该在低位买入高位卖出，这个时候就开始追跌，越跌越买，越买越跌，我真实经历的是买了，跌了我不着急，涨了才叫着急，因为一开始没有明确的目标到底什么时候抛，导致涨了被跌了还难受。

第四个阶段 策略

目前的阶段，明白了一开始就要明确买入的时候的抛出的位置，然后，死等！

交易策略的作用

交易策略的用处就是让你能够理性的决策

  1. 使用闲置的资金的多大一部分？
  2. 买什么样的股票？
  3. 什么时候买？
  4. 什么时候抛？

常见的策略

而且策略并不是越复杂越好，下面两种常见的策略都是特别简单易懂的策略，如果按照规则操作的话大概率超过大多数A股投资者的投资收益
。

定投

每个月定时买入某个品种，积少成多，集腋成裘，平滑成本，最终牛市中赚钱。
改进：通常的定投的问题在于没有设置卖出点，可以手动设置一个比如整体盈利100%或者年化20%的时候卖出。

5:5股债平衡

这就是最简单的投资策略之一：把你的资产买25%沪深300，25%中证500，50%利率债，然后定期或者定幅再平衡（三个月或者六个月操作一次，恢复各品种初始比例；或者某个品种上涨50%后触发定幅操作机制，恢复各品种初始比例）。

什么叫网格策略

网格策略也是一种简单易懂的交易策略

背景

首先，各个大的证券市场，在长时间来看，都是涨幅非常巨大的，但是这样的涨幅是通常跨越了数十年，考虑到这几十年的各种变化，像这样超级长期的持有几乎不可能，更加合理的应该是利用中短期的波动来赚取长期稳定的收益。

具体操作

主策略 网格策略基础版

1. 确定交易品种

一定是要不会死的品种，不管怎么震荡都没事，但是弄不好死了就不好了。

2. 制作网格表格
2.1. 设置交易价格

设置进入价格，进入价格的设置应该是略低于价值的时候，具体应该根据具体情况进行分析（是不是可以通过以往的价格来预测价值线在哪里）
网格大小，网格的大小根据波动的大小来，基金在通常情况下采用5%来作为网格

2.2. 设置交易金额

每个品种投入的钱都应该提前规划好，按照网格数平均分配

2.3. 设置交易日期

交易日期主要通过挂单的方式进行，等待走势落入网中

3. 压力测试

压力测试即在设置投资的资金量的时候，考虑最坏的情况能够承受多大的资金损失

4. 设置交易提醒

通过挂单软件进行交易提醒和交易

但是这其实是存在一个问题的，即在2015年到2016年的暴涨中，几乎没有操作，导致错过了一波真正的行情。

子策略

为了解决这个问题，增加几个子策略

留下利润

顾名思义，即每次把利润留下来，这部分反正是赚的钱，可以一直放到里面，一直到你觉得价格远大于价值的时候在抛售。
另外，如果觉得只留利润太少的话，在可承受的范围内可以留1倍或者多倍利润的本金（在上述做法的基础上增加杠杆，提升利润空间）

逐渐加码

考虑到价格越低的时候，与价值的差值绝对值越大，在价格越低的网格中增加投入比例，比如每个低价格的网格增加10%的投入

多网

考虑到上述最根本的问题，即为什么吃不到最大的一波行情？其实是因为网格的跨度小了了原因，假设你网格的买入卖出价格设定为20%甚至60%的差值，这个时候操作的次数更少，但是每次操作的回报率要高很多了
